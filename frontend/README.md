# Frontend (Next.js)

## Setup

```bash
# Install dependencies
npm install
```

## Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
npm start
```

## Features

- Next.js 14 with App Router
- React 18
- ESLint configured
- Health check display from backend API
